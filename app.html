      Deploy to GitHub Pages

      1. Export your site. Click Export ZIP and unzip the download. You'll get index.html plus css/ and js/ folders.
      2. Create a repository. On GitHub, create a new public repo (e.g. my-portfolio).
      3. Push the files. Add the exported files to the repo root and push: 
        git init
git add .
git commit -m "My portfolio"
git branch -M main
git remote add origin https://github.com/USER/my-portfolio.git
git push -u origin main
      4. Enable Pages. Repo → Settings → Pages → Source: Deploy from a branch → Branch: main / root → Save.
      5. Visit your site at https://USER.github.io/my-portfolio/ (allow a minute for the first build).
